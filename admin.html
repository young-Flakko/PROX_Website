Setup Checklist

// Quick-reference for wiring up the Firebase backend

  1. Create a Firebase project at console.firebase.google.com
  2. Enable Email/Password authentication
  3. Create a Firestore database (test mode to start)
  4. Paste your config values into js/firebase-config.js
  5. Sign up, then manually set your Firestore user's role to "admin"
  6. Create a Cloudinary unsigned upload preset and fill in the config
  7. Add Firestore Security Rules before going live (see comments in firebase-config.js)
